API

Атрибуты

asmodel
Путь к модели относительно контекста
строка
asscope
Контекст (родительский объект модели)
строка
asmodelkey
Ключ модели
строка
astemplate
Шаблон
строка со ссылками на поля объекта в формате {$value['field']}
aslist
Флаг, сообщающий Autosync, что шаблон должен быть применён к каждому элементу коллекции (для этого модель должна являться коллекцией, то есть содержать целочисленные поля, а также целочисленное поле length. Также задаёт первый и последний элементы для вывода в виде диапазона
Отсутствует, либо строка в формате "число1:число2"
asjson
Флаг, сообщающий Autosync, что метод pull должен интерпретировать данные элемента как JSON (метод pull работает только для элементов <input> и <textarea>
Может принимать значение "pretty", "pretty:<n>" или "pretty:<str>" для аккуратного вывода JSON, где n - ширина отступа в пробелах, а str - произвольный префикс для каждого уровня отступа.
Отсутствует, либо строка
asplaceholder
HTML-заполнитель или CSS-селектор элемента с заполнителем внутри
HTML в виде строки либо строка в формате $([ selector ]), где [ selector ] - валидный CSS-селектор
asif
Условие вывода элемента в виде JS (поддерживаются специальные переменные $value, $index, $now)
asclasses
Списки классов для добавления к элементу в зависимости от истинности условия в формате [ cond ] ? [ list1 ] | [ list2 ] или [ cond ] ? [ list1 ]
asstyles
Списки CSS правил для добавления к элементу в зависимости от истинности условия в формате [ cond ] ? [ list1 ] | [ list2 ] или [ cond ] ? [ list1 ]
asattrs
Списки атрибутов для добавления к элементу в зависимости от истинности условия в формате [ cond ] ? [ list1 ] | [ list2 ] или [ cond ] ? [ list1 ]
as-on*
Обработчик события. Внутри обработчика доступны переменные event, value, scope, index (если value не является элементом коллекции, то index равен -1)
